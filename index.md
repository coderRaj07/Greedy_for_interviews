# Greedy and Heap

Greedy algorithms make the **locally optimal choice at each step** with the hope of reaching a globally optimal solution.
Heaps (priority queues) are often used with greedy strategies when we need to **efficiently select the best candidate dynamically**.

This section covers common greedy and heap-based problem patterns frequently asked in coding interviews.
