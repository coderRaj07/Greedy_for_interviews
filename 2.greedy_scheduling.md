# 1️⃣ Job Sequencing Problem

[https://www.geeksforgeeks.org/problems/job-sequencing-problem](https://www.geeksforgeeks.org/problems/job-sequencing-problem)

### Problem

Each job has

```
deadline
profit
```

You can perform **one job per unit time**.

Goal:

```
maximize total profit
```

---

### Greedy Insight

Always try to **schedule highest profit job first**.

```
Sort jobs by profit descending
Place job in latest available slot before deadline
```

---

### Python

```python
def JobScheduling(jobs,n):

    jobs.sort(key=lambda x:x[2],reverse=True)

    max_deadline=max(job[1] for job in jobs)

    slot=[False]*(max_deadline+1)

    count=0
    profit=0

    for id,dead,p in jobs:

        for j in range(dead,0,-1):

            if not slot[j]:

                slot[j]=True
                count+=1
                profit+=p
                break

    return [count,profit]
```

---

### Java

```java
class Solution {

    int[] JobScheduling(Job arr[], int n){

        Arrays.sort(arr,(a,b)->b.profit-a.profit);

        int maxDeadline=0;

        for(Job j:arr)
            maxDeadline=Math.max(maxDeadline,j.deadline);

        boolean[] slot=new boolean[maxDeadline+1];

        int count=0;
        int profit=0;

        for(Job j:arr){

            for(int d=j.deadline;d>0;d--){

                if(!slot[d]){

                    slot[d]=true;
                    count++;
                    profit+=j.profit;
                    break;
                }
            }
        }

        return new int[]{count,profit};
    }
}
```

# 2️⃣ Task Scheduler

[https://leetcode.com/problems/task-scheduler/](https://leetcode.com/problems/task-scheduler/)

### Problem

Given tasks like:

```
A A A B B B
```

Cooldown = 2

We cannot execute same task within **2 intervals**.

---

### Key Insight

The **most frequent task controls the schedule**.

Formula:

```
(maxFreq - 1) * (cooldown + 1) + numMaxFreq
```

---

### Python

```python
from collections import Counter

class Solution:
    def leastInterval(self, tasks, n):

        freq = Counter(tasks).values()

        maxFreq = max(freq)
        maxCount = list(freq).count(maxFreq)

        return max(len(tasks),(maxFreq-1)*(n+1)+maxCount)
```

# 3️⃣ Course Schedule III (Amazon Favourite)

[https://leetcode.com/problems/course-schedule-iii/](https://leetcode.com/problems/course-schedule-iii/)

### Problem

Each course:

```
duration
lastDay
```

Goal:

```
take maximum courses
```

---

### Greedy Idea

```
Sort by deadline
Take course
If total_time > deadline → remove longest course
```

Heap stores **longest duration**.

---

### Python

```python
import heapq

class Solution:
    def scheduleCourse(self, courses):

        courses.sort(key=lambda x:x[1])

        total=0
        heap=[]

        for dur,last in courses:

            total+=dur
            heapq.heappush(heap,-dur)

            if total>last:

                total+=heapq.heappop(heap)

        return len(heap)
```

# 4️⃣ Minimum Arrows to Burst Balloons

[https://leetcode.com/problems/minimum-number-of-arrows-to-burst-balloons/](https://leetcode.com/problems/minimum-number-of-arrows-to-burst-balloons/)

### Visual

```
[1,6]
[2,8]
[7,12]
```

Arrow at **6** bursts first two.

---

### Trick

Same as interval scheduling:

```
Sort by end
Shoot arrow at end
```

---

### Python

```python
class Solution:
    def findMinArrowShots(self, points):

        points.sort(key=lambda x:x[1])

        arrows=1
        end=points[0][1]

        for s,e in points:

            if s>end:
                arrows+=1
                end=e

        return arrows
```
